# Pdman Web

## 安装

`#`表示执行命令
```shell script
# git clone https://github.com/whaty/MARTIN-ERD.git pdman_web

# cd pdman_web

# yarn install
```

## 运行

```
# yarn start
```
打开 [http://localhost:3000](http://localhost:3000)

浏览器中可以看到欢迎页，就说明成功启动了。
